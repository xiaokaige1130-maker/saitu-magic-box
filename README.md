# 晒图魔方

Windows 原生 EXE 图片整理工具，用于摄影/电商图片分类、重复识别、AI 初筛和人工复核。

## 功能

- 扫描本地图片文件夹，支持 jpg、png、bmp、webp、tif 等常见格式
- 识别完全重复、文件名副本、视觉相似图片
- 自动生成缩略图、质量分、清晰度、曝光、反差、饱和度指标
- 自动分类为“精选、较差、空镜、废片”，并生成问题标签
- 支持 10%、20%、50%、80% 推荐选优
- 支持卡片内快速改分类，键盘 1-5 设置星级
- 支持复制精选、废片、重复候选、组内最佳到输出文件夹
- 支持勾选任意图片后单独导出
- 支持按分类整理；原有 Web 接口仍保留 CSV 导出能力
- 默认不删除、不移动原图

## 桌面操作

- `开始筛选`：扫描原图文件夹并自动分类
- `全选当前` / `清空勾选`：管理当前列表的勾选状态
- `重复去重`：只显示每个重复/相似组里系统判断最好的那一张
- `勾选去重结果`：勾选当前去重结果，或在其它视图勾选全库去重后的保留图
- `导出勾选`：只复制勾选图片到输出目录
- `导出去重后全部`：全库去重，每组只保留最佳一张，唯一图片照常保留
- `导出当前分类`：复制当前视图内的图片
  - 当前视图为“重复去重”时，只导出去重后的结果，不会把重复成员全导出
- `按分类导出全部`：按精选、较差、空镜、废片分别复制
- 卡片底部的“精选、较差、空镜、废片”按钮可手动改分类
- 双击缩略图可打开原图查看

## 开发启动

```powershell
cd C:\Users\云电脑\image-cube
python launcher.py
```

## 默认扫描目录

```text
C:\Users\云电脑\Desktop\白底图\外贸绞肉机图
```

## 打包 EXE

```powershell
cd C:\Users\云电脑\image-cube
.\build-exe.ps1
```

生成位置：

```text
C:\Users\云电脑\image-cube\dist\晒图魔方\晒图魔方.exe
```

打包后的程序直接打开 Windows 桌面窗口，不启动浏览器页面。

## RAW 文件

代码已经识别 cr2、cr3、nef、arw、raf、dng 扩展名。当前默认依赖不强制安装 `rawpy`，因此没有 RAW 解码环境时这些文件会显示在扫描失败列表里；安装可用的 `rawpy` 后会自动读取 RAW 预览。

## 原则

- 不删除原图
- 不移动原图
- 只生成本地数据库、缩略图和筛选报告
